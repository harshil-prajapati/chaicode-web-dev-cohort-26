# Dev Tool Landing Page - Cursor Clone

## Overview
This project is a complete clone of the Cursor IDE landing page built using pure HTML and CSS.
It is created as part of the Web Dev Cohort 2026 assignment.

The landing page features a modern dark theme design with responsive layouts, hero sections,
testimonials, feature showcases, and a comprehensive footer.


## Project Structure
```
dev-tool-landing-page-cursor/
├── index.html          # Main HTML file
├── styles.css          # All styling
├── README.md           # Project documentation
└── assets/
    ├── avatar/
    ├── card-placeholder/
    ├── hero-container/
    ├── logo/
    └── output/
```

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- No JavaScript (Static design only)

## Setup Instructions
1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. No build process or dependencies required
